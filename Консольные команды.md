# Консольные команды

## Создать новый проект
```
cargo new
```

## Проверка на ошибки

Производится проверка на ошибки, но не создается исполняемый файл.
```
cargo check
```
## Билд проекта

```
cargo build
```

Бинарник помещается в папку `target/debug`

Для получения релизной версии запускается команда:
```
cargo build --release
```

## Запуск тестов

```
cargo test
```

## Документация по пакетам

Если вы подгружены зависимости, то есть возможность скачать документацию по этим пакетам и открыть их локально в браузере. 
```
cargo doc --open
```
